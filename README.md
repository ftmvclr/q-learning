This program simulates q-learning process of an agent on a 10x10 grid map, with a single load to pick up  
and a single square to put it to. The coefficients and limitations can be tweaked inside the source code and be experimented on.  
For instance: a more drastic decrease in epsilon value should help optimize the solution and converge much quicker than a mild decline.  
For now the gamma value is 0.95 whereas the alpha value is 0.1. I plan on improving the code to allow multiple loads and multiple places  
hence the data structures for their coordinates specifically.
